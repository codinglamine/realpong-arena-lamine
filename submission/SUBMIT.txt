TOURNAMENT SUBMISSION
=====================

Submit as:   lamine.py:lamine.pt               (regular arena)
             lamine_chaos.py:lamine_chaos.pt   (chaos arena)

Files (keep each pair together):
  lamine.py / lamine.pt              - regular arena: ~1.7M-param CNN (model + Agent class)
  lamine_chaos.py / lamine_chaos.pt  - chaos arena: CNN

How the arena uses it:
  It imports the .py, builds Agent("...pt"), then calls agent.reset() at the start of
  each game and agent.act(frame) every step. frame = 80x80 binary image, your paddle on
  the RIGHT. act() returns 2 (UP) or 3 (DOWN).

Dependencies: numpy + torch only (self-contained, no local imports).

Test locally:
  python arena.py       lamine.py:lamine.pt              willem-cnn.py:willem-cnn.pt bf
  python arena_chaos.py lamine_chaos.py:lamine_chaos.pt  willem-cnn.py:willem-cnn.pt bf
